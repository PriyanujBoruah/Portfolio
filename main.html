Advanced Emotion Analysis Platform

This project is a full-stack web platform for real-time emotion detection in text. It leverages the power of pre-trained transformer models from Hugging Face to analyze user-provided text and instantly display the probabilities of various emotions. The application provides a user-friendly interface, real-time analysis, user authentication, a history of past analysis, and a responsive UI.

Supports 18 distinct emotions and achieved 77.52% accuracy on a test set of 21,468 sentences.

Recipe Recommendation Web Application

This project is web application that helps users find recipes based on their preferred ingredients. It's built using Python (Flask, Pandas) for the backend and HTML, CSS and JavaScript for the frontend. The applications provides a user-friendly interface for searching recipes based on ingredients and browsing random recipes suggestions.

Advanced Sudoku Solver

This project is a comprehensive Sudoku Solver application built using Python and Streamlit, implementing an efficient backtracking algorithm; with the option for solving a sudoku from user input or generate multiple random sudokus for benchmarking.

Successfully tested and validated against 6,482 Sudoku puzzles, solving the entire set in 5.62 seconds with an average time of 0.00087 seconds per puzzle; ensuring performance, reliability and accuracy in performance.
